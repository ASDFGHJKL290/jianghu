## 三、人际关系系统

### 3.1 设计目标

NPC不再只是知识问答机器人，而是用户江湖路上的**引路人、盟友或对手**。关系越深，NPC的称呼、语气、解锁内容都会随之变化。

### 3.2 关系维度（三维模型）

| 维度 | 说明 | 影响 |
|------|------|------|
| **亲密度** | 数值 0-100，反映情感亲近程度 | 称呼、语气、是否透露秘密 |
| **声望** | 正/邪分立，正声望+正值，邪声望-负值，0为中立 | 能否触发某些任务、解锁某些NPC、影响战斗选项 |
| **了解度** | NPC对用户背景的了解程度（0-5星） | NPC是否记住用户的身世、门派、承诺 |

### 3.3 亲密度等级与称呼（全NPC版）

| 等级 | 分值区间 | 店小二 | 武林盟主 | 神秘大侠 | 扫地僧 | 洪七公 | 任盈盈 | 风清扬 |

| 初识 | 21-40 | 客官您又来啦！ | 壮士请讲 | …… | 施主 | 嘿，小子 | …… | …… |
| 相熟 | 41-60 | 少侠！/兄弟！ | 壮士 | …… | 少侠 | 小兄弟！ | 少侠 | 少侠 |
| 信任 | 61-80 | 哎哟自己人！ | 少侠 | …… | 居士 | 娃娃 | 少侠做得 | …… |

**称呼触发场景：**
- 每次对话开头，NPC用当前等级称呼打招呼
- 关键剧情点可临时触发特殊称呼（如任务接受/拒绝时的临时称呼）

### 3.4 了解度机制

系统从对话中提取用户信息（身世、门派、目标、仇人），存储在 `relation.json` 的 `known_facts` 字段中。了解度≥3星时，NPC主动提及用户背景：

> 店小二："兄弟，听说你是少林寺出来的？少林的功夫……哎，那可是真本事啊！"

```
known_facts = {
    "origin": "少林寺弃徒",
    "goal": "夺回屠龙刀",
    "enemy": "青城派",
    "weapon": "长剑"
}
```

## 四、战斗系统

### 4.1 设计理念

战斗是**选项式对话战斗**，不需要写复杂数值引擎，靠LLM判断剧情走向 + 选择分支，给用户「真实武侠过招」的代入感。

**核心理念：**
- 不是"打怪掉血"，而是「文斗 + 策略选择」
- 战斗结果由：战斗选择 + 武功等级 + 声望阵营 + 装备道具 共同决定
- 同一场战斗，正邪声望下可选择不同打法
- 战斗失败≠游戏结束，可以认输、和解、逃跑

### 4.2 战斗触发场景

| 触发来源 | 示例 | 战斗类型 |
|----------|------|----------|
| 任务要求 | 护送证人途中遭遇山贼 | 任务战斗 |
| 正邪冲突 | 撞见欧阳克欺负人（帮OR不帮） | 选择触发 |
| 江湖偶遇 | 行走江湖时遇到劫匪 | 随机遭遇 |
| NPC挑衅 | 风清扬考验剑法 | 师徒切磋（必败引导教程） |
| 限时任务 | 每日华山擂台 | 答题+选项战斗 |
| 门派恩怨 | 少林篇中挑战少林俗家弟子 | 门派试炼 |

### 4.3 战斗流程

```
【战斗开始】
    ↓
① 系统播报战斗场景（LLM生成战斗开场白）
    ↓
② 展示对手信息：姓名、阵营、难度（⭐~⭐⭐⭐⭐⭐）
    ↓
③ 展示用户当前状态：声望（正/邪）、武功等级、可用装备
    ↓
④ 展示战斗选项（2-4个，每次选1个）
    ↓
⑤ LLM根据选项 + 用户属性 → 判定结果（胜/负/平/触发特殊）
    ↓
⑥ 战斗结算：奖励/惩罚/剧情分支
    ↓
【战斗结束】
```

### 4.4 战斗属性体系

| 属性 | 说明 | 影响 |
|------|------|------|
| **武功等级** | 0-100，初始10，每次战斗/任务提升 | 影响战斗选项成功率 |
| **声望** | 正/邪分立，影响可解锁的战斗选项 | 正声望可"劝降"，邪声望可"威慑" |
| **装备加成** | 武器提供成功率加成 | 倚天剑+50%，金丝刀+20% |
| **消耗品** | 战斗前可用（少林大还丹等） | 临时提升成功率 |
| **NPC援助** | 亲密度高的NPC可在战斗后协助 | 某些选项解锁"请XX帮忙" |

武功等级提升方式：
- 完成战斗胜利 +10~+30
- 完成传说任务 +50
- 获得武功秘籍道具（独孤九剑剑谱等）+20~+50
- 每日问答任务答对 +5

### 4.5 战斗选项设计

每场战斗提供2-4个选项，根据用户属性解锁：

```
【基础选项】（始终可用）
  A. 正面硬刚 — 硬碰硬，最直接
     成功率：武功等级 / (武功等级 + 对方难度×10) × 100%
  
  B. 观察破绽 — 先看清对方套路再出手
     成功率：武功等级 / (武功等级 + 对方难度×8) × 100%
     成功后解锁"精准打击"选项（伤害×1.5）

【正派声望解锁】（声望≥20正）
  C. 以德服人 — 劝对方放下武器
     成功率：声望值 / (声望值 + 对方难度×5) × 100%
     成功：完胜，获得声望+5，物品掉落

【邪派声望解锁】（声望≤-20邪）
  C. 以强凌弱 — 亮出实力震慑对方
     成功率：武功等级 + 30
     成功：速胜，获得物品，声望-5（更邪）

【装备加成解锁】（持有对应武器时）
  D. 以X制敌 — 使用倚天剑/血刀等特殊武器
     成功率：基础+装备加成（见道具表）
     特殊效果：部分敌人对特定武器有弱点

【道具解锁】（持有对应消耗品时）
  E. 使用XXX — 使用消耗品改变战局
     成功率：固定（如大还丹→+50%）

【NPC援助解锁】（对应NPC亲密度≥60时）
  F. 请XXX相助 — 呼叫NPC助战
     成功率：亲密度 / 100 + 50%
     额外效果：NPC亲密度+2，战斗后触发NPC特殊对话

【逃跑选项】（始终可用）
  G. 三十六计走为上 — 直接撤退
     成功率：80%
     后果：声望-3，任务可能失败
```

### 4.6 战斗结果类型

| 结果 | 触发条件 | 效果 |
|------|----------|------|
| **大胜** | 选择正确 + 成功率>80% | 武功+30，声望+10（正）或+5（邪），道具掉落 |
| **险胜** | 选择正确 + 成功率40-80% | 武功+15，声望+5（正）或+2（邪） |
| **平局** | 成功率<40% | 双方停手，武功+5，无惩罚 |
| **惜败** | 选项正确但骰点差 | NPC提出"认输"或"再战"选项 |
| **完败** | 选择错误或骰点极差 | 声望-10，触发平一指救人剧情 |
| **逃跑** | 选择逃跑 | 声望-3，任务标记失败，可重新挑战 |
| **劝降** | 正声望+说服选项 | 完胜变种：不战而胜，奖励翻倍，声望+15 |

## 五、任务系统（增强版）

### 5.1 任务分类（五类）

| 任务类型 | 触发方式 | 代表任务 |
|----------|----------|----------|
| **传闻任务** | 关键词触发（RAG+关键词双重） | 初入江湖、侠客行、门派风云 |
| **委托任务** | NPC主动提出（亲密度≥40解锁） | 送信、寻物、打探消息 |
| **限时任务** | 每日重置，每日3个 | 华山擂台、江湖传闻、帮派日常 |
| **传说任务** | 多步链式任务，前置完成后解锁 | 初入江湖→九阴真经→武林至尊 |
| **奇遇任务** | 随机概率触发 | 桃花岛偶遇、扫地僧考验 |

### 5.2 传说任务链（一天可完成版）

```
【初入江湖】（传闻任务）⭐ ——约10分钟
  触发：关键词"江湖""闯荡""初来乍到"
  内容：回答3道江湖基础知识（门派/武功各1题）
  奖励：声望+10（正），武功+10，初始道具【江湖地图碎片×1】

          ↓ 完成

【侠客行·名剑篇】（传闻任务）⭐⭐ ——约10分钟
  触发：关键词"剑""倚天""屠龙""神兵"
  内容：向店小二打听名剑故事，选择感兴趣方向（少林/武当/华山）
```

### 7.1 设计目标

地图不是开放世界探索，只是**展示用户当前所在地点**，配合地点图标和简单导航，增加代入感。

### 7.2 实现方式

顶部一行地点图标栏（类似快捷方式），点击切换：

```
[悦来酒楼] —[武林盟主府] —[华山之巅] —[少林寺] —[武当山] —[桃花岛] —[黑木崖] —[???]
```

- 前3个地点初始解锁，后5个随任务/声望逐步解锁
- 点击地点 → 切换NPC对话场景 → NPC变成该地点的NPC
- 未解锁地点显示为「???」，点击提示解锁条件
