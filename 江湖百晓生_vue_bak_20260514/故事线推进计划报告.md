### 1.2 核心问题

1. **任务量太少**：只有5个任务，玩家10分钟就通关，无可玩性
2. **无故事主线**：任务是孤立的"知识问答"，没有"为什么要做这个任务"的动机
3. **NPC无故事角色**：每个NPC只是"可以触发关键词"，但没有嵌入故事线
4. **门派无剧情差异**：选少林和选丐帮，后续对话完全一样
5. **无结局系统**：做到最后一个任务就结束了，没有"江湖结局"
6. **任务推进无叙事反馈**：任务完成了，但故事没有推进感

### 1.3 现有任务系统机制

```
玩家说话 → 检测关键词 → 匹配任务触发条件（NPC权限+前置任务+关键词）
      ↓
任务激活 → 玩家对话匹配步骤关键词 → 步骤完成
      ↓
所有步骤完成 → 任务完成 → 发奖励 → 解锁下一任务
```

**关键文件：**
- `prompts.py`：ALL_QUESTS字典，定义5个任务的trigger/title/steps/reward
- `chat.py`：`check_quests()` 检测任务触发，`_advance_quest_step()` 推进步骤
- `game.py`：教程系统与任务系统部分耦合（初入江湖完成→推进教程）
- `main.py`：/api/chat 端点处理任务完成后的奖励发放

## 二、故事线设计方案

### 2.1 主线：九阴真经的千年秘密

> **核心设定**：九阴真经不是武功秘籍，而是一本"修心指南"。得经者不是得到天下武学总纲，而是得到"如何修心"的方法。黄裳创作九阴真经，不是因为他武功高强，而是因为他领悟了"武 = 止戈 = 修心"。玩家在悦来客栈醒来，失去记忆，身上只有一张残破地图，由此展开江湖历险。

```python
# chat.py — check_quests() 修改（约第147行）
def check_quests(self, user_input: str, npc_name: str, user_id: str = "default") -> list:
    user_lower = user_input.lower()
    candidates = []
    # 获取当前章节
    chapter_prog = self.get_chapter_progress(user_id)
    current_chapter = chapter_prog["current_chapter"]
    # 获取玩家门派（从game.py的全局变量）
    import game as g
    player_faction = g.faction or "无门派"
    
    for qid, q in ALL_QUESTS.items():
        # --- 原有过滤条件 ---
        if npc_name not in q["npc_permission"]:
            continue
        if self.quest_completion.get(user_id, {}).get(qid):
            continue
        prereq = q.get("prerequisite")
        if prereq and not self.quest_completion.get(user_id, {}).get(prereq):
            continue
        if any(quest.get("qid") == qid for quest in self._user_quests(user_id)):
            continue
        
        # --- 新增：章节过滤 ---
        if q.get("chapter", 1) != current_chapter:
            continue  # 只显示当前章节的任务
        
        # --- 新增：门派分支 ---
        faction_branch = q.get("faction_branch", {}).get(player_faction)
        if faction_branch:
            # 使用门派专属steps
            q = {**q, "steps": faction_branch["steps"], "reward_data": faction_branch.get("reward_data", q["reward_data"])}
        
        matched = [kw for kw in q["trigger"] if kw.lower() in user_lower]
        if matched:
            candidates.append((len(matched), qid, q))
    
    if not candidates:
        return []
    candidates.sort(reverse=True, key=lambda x: (x[0], len(x[2]["trigger"])))
    best_qid, best_q = candidates[0][1], candidates[0][2]
    return [{"qid": best_qid, **best_q}]
```

### 10.3 LLM输出格式

NPC对话末尾输出 `{"gift": {...}}`，后端解析后**真送进背包**。

```
示例（店小二）：
"别气，今日请你喝酒，不醉不归！"
{"gift": {"name": "女儿红", "icon": "🍶", "count": 1, "reason": "安慰你"}}

示例（完成任务）：
"干得漂亮！这是赏你的！"
{"gift": {"name": "打狗棒法入门", "icon": "📖", "count": 1, "reason": "任务奖励"}}
```

**格式规则**：
1. `{"gift": {...}}` 必须单独一行，在回复文本之后
2. 后端解析后从回复文本中去掉此JSON（玩家只看到对话）
3. `reason` 字段用于前端弹窗提示

```python
# chat.py — 新增方法（在 _strip_json_suffix() 之后）

def _extract_gift(self, text: str) -> dict | None:
    """从LLM输出中提取 gift JSON"""
    import re
    # 支持单双引号
    pat1 = r'\{\s*"gift"\s*:\s*\{[^}]+\}\s*\}'
    pat2 = r"\{\s*'gift'\s*:\s*\{[^}]+\}\s*\}"
    for pat in [pat1, pat2]:
        m = re.search(pat, text)
        if m:
            try:
                return json.loads(m.group())
            except:
                try:
                    return eval(m.group())  # fallback for single quotes
                except:
                    pass
    return None

def _strip_gift(self, text: str) -> str:
    """去掉文本末尾的 gift JSON"""
    import re
    text = re.sub(r'\{\s*"gift"\s*:\s*\{[^}]+\}\s*\}\s*', '', text)
    text = re.sub(r"\{\s*'gift'\s*:\s*\{[^}]+\}\s*\}\s*", '', text)
    return text.strip()
```

## 八、武侠理念说明（新增）

> **核心理解**：武 = 止戈 = 修心
> 本游戏的所有剧情设计，都应基于这一理解。

### 8.1 武 = 止戈

**错误理解**：武 = 力量 = 我可以打败任何人 = 我可以为所欲为

**正确理解**：武 = 止戈 = 停止战争/冲突 = 让天下太平

**游戏设计体现**：
- 九阴真经不是"武功秘籍"，而是"修心指南"
- 任我行不是"反派"，而是"误解了武"的可怜人
- 最终任务不是"打败任我行"，而是"帮助他领悟武 = 修心"

---

### 8.2 修心，修的是什么？

#### 8.2.1 修的是"对力量的理解"

| 错误认知（任我行） | 正确认知（修心后） |
|----------------|----------------|
| 武 = 力量 = 我可以打败任何人 | 武 = 止戈 = 让战争/冲突停止 |
| 力量越大，越可以为所欲为 | 力量越大，责任越大，越不能轻易使用 |
| 真正的高手，需要证明自己 | 真正的高手，不需要证明自己 |

**修心过程**：从"我想变强"到"我明白强是为了不用"

---

#### 8.2.2 修的是"对自我的执着"

| 错误认知（任我行） | 正确认知（修心后） |
|----------------|----------------|
| 我练武，是为了"我"——我的名声、我的地位、我的快感 | 我练武，是为了"止戈"——保护他人、平息争斗、维持和平 |
| 练武是为了满足自我 | 武学的终极目标，是让自己不再需要出手 |

**修心过程**：从"我为武"到"武为我（止戈）"

---

#### 8.2.3 修的是"对暴力的态度"

| 错误认知（任我行） | 正确认知（修心后） |
|----------------|----------------|
| 暴力是解决问题的手段 | 暴力是失败，不是成功 |
| 打服别人，就是"赢" | 真正的高手，能用非暴力手段解决问题 |
| | "不战而屈人之兵"才是最高境界 |

**修心过程**：从"用拳头说话"到"不用拳头也能解决"

---

#### 8.2.4 修的是"德"（仁、义、礼、智、信）

> **关键**：武德不是附加在"武"上的道德，而是"武"的本身。

| 德目 | 含义 | 体现 |
|------|------|------|
| 仁 | 真正的武者，不会滥杀无辜 | 不杀无辜，不伤平民 |
| 义 | 真正的武者，会为正义而战（但"战"是最后手段） | 不为私利而战，只为止戈而战 |
| 礼 | 真正的武者，尊重对手，不恃强凌弱 | 尊敬对手，不用卑鄙手段 |
| 智 | 真正的武者，明白"武 = 止戈"，不会轻易动手 | 能用智慧解决的，不用武力 |
| 信 | 真正的武者，言出必行，但不会用武力强迫别人 | 信守承诺，但不强求别人信守 |

**修心过程**：从"我有力量所以我说了算"到"我有力量所以我更要谨慎"

---

### 8.3 品质是"果"，不是"因"

> 谦逊、尊敬、豁达、对武的敬畏——这些品质是"修心"的**结果**，不是**原因**。

| 品质 | 是"因"还是"果" | 说明 |
|------|-------------------|------|
| 谦逊 | 果 | 因为明白了"武 = 止戈"，所以自然谦逊 |
| 尊敬 | 果 | 因为明白了"武"不是欺负弱者，所以尊敬对手 |
| 豁达 | 果 | 因为明白了"武"的目的不是赢，所以不执着胜负 |
| 对武的敬畏 | 果 | 因为明白了"武 = 止戈"的分量，所以敬畏 |

**真正的"因"**：明白"武 = 止戈 = 修心"

**这些品质是自然流露**，不是强行外加的。

---

### 8.4 任我行的"心病"

**病症**：任我行练了一辈子武，但越来越痛苦。

**病因**：他一直在"修错心"——
- 他修的是"如何让力量更大"
- 他修的是"如何打败所有人"
- 他修的是"如何让自己可以为所欲为"

**治疗方法**：帮助他明白"武 = 止戈 = 修心"

**游戏设计体现**：
- 任我行不是"反派"，而是"病人"
- 玩家的任务不是"打败"他，而是"治疗"他
- 最终结局不是"任我行被打败"，而是"任我行被感化"

---

### 8.5 九阴真经的真正秘密

**错误理解**：九阴真经是"武功秘籍"，得经者可得天下武学总纲。

**正确理解**：九阴真经是"修心指南"，得经者可得"如何修心"的方法。

**黄裳创作九阴真经的原因**：
- 不是因为他武功高强
- 而是因为他领悟了"武 = 止戈 = 修心"
- 他写下经书，是为了让后人不要再误解"武"

**游戏设计体现**：
- 九阴真经的内容，是关于"如何修心"的指南
- 不是"如何变强"，而是"如何理解力量"
- 玩家通过集齐残页，逐步领悟"武 = 止戈 = 修心"

---

### 8.6 如何体现在游戏设计中？

#### 8.6.1 任务设计

| 任务 | 错误设计 | 正确设计 |
|------|---------|---------|
| 任务15（第一次抉择） | 交出残页 vs 继续追寻真相 | 如何用九阴真经帮助江湖？（感化/传道/传承） |
| 任务20（任我行的质问） | 任我行来袭，玩家需要应战（BOSS战） | 任我行质问："我练武一辈子，为什么越来越痛苦？"玩家需要帮助他 |
| 任务29（最终任务） | 最终BOSS战（任我行，HP 200） | 用九阴真经的修心之法感化任我行 |

---

#### 8.6.2 结局设计

| 结局 | 错误设计 | 正确设计 |
|------|---------|---------|
| A | 侠义结局（任我行被感化，玩家成为守护者） | 修心结局（帮助任我行领悟"武"的真谛，一同归隐） |
| B | 霸道结局（击败任我行，成为武林盟主） | **删除**（不符合"武 = 止戈"的理念） |
| C | 隐退结局（销毁九阴真经，与任盈盈归隐） | 归隐结局（销毁九阴真经，让江湖不再为它争斗） |
| D | 自由结局（自创武学，开创全新武学时代） | 传道结局（自创"江湖诀"，开宗立派，传授"武 = 修心"的理念） |

---

#### 8.6.3 NPC对话设计

**错误设计**：NPC说"我教你武功，让你变强"

**正确设计**：NPC说"我教你修心，让你明白武 = 止戈"

**示例**：
- 扫地僧："施主，老衲教你易筋经，不是让你变强，而是让你明白'武 = 修心'。"
- 风清扬："独孤九剑的精髓，不是'打败对手'，而是'后发先至，以柔克刚'——这才是'止戈'的真谛。"
- 洪七公："降龙十八掌的精髓，不是'天下无敌'，而是'侠之大者，为国为民'。"

---

### 8.7 总结

**武侠游戏的核心**：不是"让玩家变强"，而是"让玩家明白武 = 止戈 = 修心"。

**任我行的意义**：他是"误解了武"的典型，他的痛苦提醒玩家——"武"不是力量，而是责任。

**九阴真经的意义**：它是"修心指南"，不是"武功秘籍"。玩家通过集齐残页，逐步领悟"武 = 止戈 = 修心"。

**所有剧情设计的原则**：
1. 没有"霸道结局"（称霸不是武）
2. 任我行不是反派，而是"病人"
3. 最终任务不是"打败任我行"，而是"帮助他领悟"
4. NPC对话要体现"武 = 止戈 = 修心"的理念
