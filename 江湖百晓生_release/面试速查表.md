# 面试速查表 — 江湖百晓生

## 一句话介绍
> 自研 AI NPC 叙事引擎，13 个 NPC 在武侠世界里自由交互，10000+ 行 Python，45 天独立完成。

## 技术栈
FastAPI + Vue3 + DeepSeek + ChromaDB + BGE-large-zh + LangChain + NetworkX + KataGo

## 核心架构（4 个模块）

### 1. 对话流程
```
用户发 POST /api/chat → main.py 路由 → chat.py build_chat_response()
→ 组装 prompt（NPC人格 + 情感 + 记忆 + 背包 + 任务 + 天气 + 知识库）
→ DeepSeek API → 解析回复 → 更新情感/记忆/好感度
```

### 2. 三层过滤管道（事件引擎）
```
DeepSeek 批量生成 → ChromaDB 语义去重(cosine>0.75) → NetworkX 知识图谱校验
→ AI 评审团评分(4维度≥5分) → 入库 SQLite | 通过率约64%
```

### 3. 记忆系统（三层 + 时间衰减）
- 日常: 每次对话记录，24h 无引用衰减
- 罕见: 引用2+次升级到此，降级前先引用
- 稀有: 重要剧情绑定，频繁引用保留
- 升降级: 引用频率↑升级，沉默期↓降级删除

### 4. 事件链系统
```
事件触发 → condition_edges匹配 → 多候选链打分(地点/好感度/连续性)
→ 选中链下一节点 → 自动注册为任务 → 注入NPC对话prompt
```

## RAG 部分
- 知识库: `江湖奇闻轶事知识库.md` → chunk_size=512, overlap=50 → ChromaDB
- 检索: 用户消息>8字符 → similarity_search(q, k=2) → 拼进prompt当【江湖见闻】
- 没做rerank/混合检索: k=2已全量，量级小不需要
